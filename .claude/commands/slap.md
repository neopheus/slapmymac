Interact with SlapMyMac via its local MCP server.

Usage:
- `slap status` — show current slap count and app state
- `slap stats` — show slap statistics
- `slap trigger [mode]` — play a sound (pain/sexy/halo)
- `slap mode [mode]` — switch voice pack

Execute the appropriate curl command against http://localhost:7749 and show the JSON response formatted.
